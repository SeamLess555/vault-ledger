# Vault Ledger Pro

A personal finance management app built with Flutter. Organize and track your savings across multiple allocation units with automatic income splitting and expense logging.

## https://www.paypal.com/ncp/payment/6ZWB8AZ97U77N

## Features

- 💰 **Multi-Unit Allocation** - Create storage units with custom percentage allocations
- 📊 **Automatic Income Splitting** - Distribute income across units based on allocation percentages
- 📝 **Transaction History** - Complete ledger of all income and expense transactions
- 📈 **Monthly Reports** - Track monthly income and expenses at a glance
- 💾 **Local Data Persistence** - All data stored locally using SharedPreferences
- 🎨 **Material Design 3** - Modern, clean UI with dark theme support

## getting started
1. create your savings split ( I recommend the 50,30,20 for a start) by first tapping "new storage unit", then naming it and deciding how much of the allocation it will take up % wise. Add as many as necessary.
2. then tap the deposit income and enjoy the automatic allocation.
3. from there you can customize everything and you even have a history tab to track how much you spend and you can delete and edit them if you made a mistake.
